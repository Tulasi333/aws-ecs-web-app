# 🚀 AWS ECS Web Application Deployment

## 📌 Project Overview
This project demonstrates the deployment of a containerized static web application using **Docker**, **Amazon Elastic Container Registry (ECR)**, and **Amazon Elastic Container Service (ECS) with Fargate**. The application is exposed to the internet through an **Application Load Balancer (ALB)**, ensuring scalability and high availability.

## 🛠️ Architecture
GitHub → Docker → Amazon ECR → Amazon ECS (Fargate) → Application Load Balancer → Users

## 🚀 Technologies Used
- AWS ECS (Fargate)
- Amazon ECR
- Docker
- Application Load Balancer (ALB)
- IAM Roles
- VPC & Security Groups
- GitHub
